### Data:
- **Histone Modifications**: CUT&Tag data (H3K27ac, H3K4me1).
- **Methylation Data**: Whole-genome bisulfite sequencing (WGBS) for DMRs.
- **Mutational Data**: WGS for DNA sequences.
- **Control Data**: Healthy blood stem cells (CD34+).
